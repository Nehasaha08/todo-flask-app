# Todo List App (Flask)

A simple Todo List web app built using Flask, SQLite, HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## Run Locally

1. Install dependencies:
pip install -r requirements.txt

2. Run app:
python app.py

3. Open browser:
http://127.0.0.1:5000
